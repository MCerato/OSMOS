ControlBox module#

Wrap the GALIL gclib library.

Description#

Allow to connect and communicate with a specific GALIL board. The board is identified through its IP address.

Warning

The IP address of the board has to be fixed previously. (parameter DH should be 0)

Libraries/Modules#

  • gclib library (https://www.galil.com/sw/pub/all/doc/gclib/html/python.html)

    • provided by GALIL to communicate with their product

Version#

  • 1.0.0.0

Notes#

  • None
