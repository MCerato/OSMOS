gclib module#

Python wrapper for Galil gclib. Contact softwaresupport@galil.com with questions, comments, and suggestions.

exception gclib.GclibError#

Bases: Exception

Error class for non-zero gclib return codes.

class gclib.py#

Bases: object

Represents a single Python connection to a Galil Controller or PLC.

GAddresses()#

Provides a dictionary of all available connection addresses.

Returns a dictionary mapping ‘address’ -> ‘revision reports’, where possible e.g. {}

GArrayDownload(name, first, last, array_data)#

Downloads array data to a pre-dimensioned array in the controller’s array table. array_data should be a list of values (e.g. int or float)

GArrayDownloadFile(file_path)#

Downloads a csv file containing array data at file_path.

GArrayUpload(name, first, last)#

Uploads array data from the controller’s array table.

GArrayUploadFile(file_path, names=[])#

Uploads the entire controller array table or a subset and saves the data as a csv file specified by file_path. names is optional and should be a list of array names on the controller.

GAssign(ip, mac)#

Assigns IP address over the Ethernet to a controller at a given MAC address. Linux/OS X users must be root to use GAssign() and have UDP access to send on port 68.

GClose()#

Closes a connection to a Galil Controller.

GCommand(command)#

Performs a command-and-response transaction on the connection. Trims the response.

GFirmwareDownload(file_path)#

Upgrade firmware.

GInfo()#

Provides a useful connection string. Please include the output of this function on all support cases.

GInterrupt()#

Provides access to PCI and UDP interrupts from the controller.

GIpRequests()#

Provides a dictionary of all Galil controllers requesting IP addresses via BOOT-P or DHCP.

Returns a dictionary mapping ‘model-serial’ –> ‘mac address’ e.g. {‘DMC4000-783’: ‘00:50:4c:20:03:0f’, ‘DMC4103-9998’: ‘00:50:4c:38:27:0e’}

Linux/OS X users must be root to use GIpRequests() and have UDP access to bind and listen on port 67.

GListServers()#
GMessage()#

Provides access to unsolicited messages from the controller.

GMotionComplete(axes)#

Blocking call that returns once all axes specified have completed their motion.

GOpen(address)#

Opens a connection a galil controller. See the gclib docs for address string formatting.

GProgramDownload(program, preprocessor='')#

GProgramDownloadFile(file_path, preprocessor='')#

Program download from file. See the gclib docs for preprocessor options.

GProgramUpload()#

Uploads a program from the controller’s program buffer.

GProgramUploadFile(file_path)#

Program upload to file.

GPublishServer(server_name, publish, save)#
GRemoteConnections()#
GServerStatus()#

GSetupDownloadFile(file_path, options)#

Downloads specified sectors from a Galil compressed backup (gcb) file to a controller.

Returns a dictionary with the controller information stored in the gcb file. If options is specified as 0, an additional “options” key will be in the dictionary indicating the info sectors available in the gcb

GSleep(val)#

Provides a blocking sleep call which can be useful for timing-based chores.

GTimeout(timeout)#

Set the library timeout. Set to -1 to use the intitial library timeout, as specified in GOpen.

GVersion()#

Provides the gclib version number. Please include the output of this function on all support cases.

__del__()#

Destructor for the Connection class. Ensures close gets called to release Galil resource (Sockets, Kernel Driver, Com Port, etc).

__init__()#

Constructor for the Connection class. Initializes gclib’s handle and read buffer.

property timeout#

Convenience property read access to timeout value. If -1, gclib uses the initial library timeout, as specified in GOpen.
